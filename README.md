# Homebrew Tap for Milieu

This is the official Homebrew tap for **milieu**, a CLI for end-to-end encrypted dotenv sync.

- Project repo: https://github.com/citizenhicks/milieu
- Website: https://milieu.sh

## Install

```bash
brew tap citizenhicks/milieu
brew install milieu
```

## Notes

This tap is maintained by the milieu project and provides macOS and Linux bottles when available.
